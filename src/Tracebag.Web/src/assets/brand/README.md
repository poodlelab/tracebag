# Tracebag brand mark

The mark combines two product ideas:

- The **bag** is the durable incident package: logs, counters, captures, notes,
  and artifacts kept together as evidence.
- The **trace** is the live runtime signal that Tracebag observes and preserves.

`tracebag-mark.svg` is the canonical source. Product surfaces should reference
this asset directly rather than drawing a separate logo in component markup.

The dark tile is intentionally part of the mark so it stays recognizable on
light, dark, and transparent surfaces and remains legible at favicon size.
