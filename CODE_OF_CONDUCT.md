# Code of Conduct

Tracebag contributors are expected to create a professional, welcoming, and
constructive community.

## Expected behavior

- Be respectful and assume good intent while discussing ideas.
- Give technical feedback about the work, not the person.
- Welcome different backgrounds and levels of experience.
- Share context, evidence, and tradeoffs openly.
- Respect privacy, security reports, and confidential diagnostic data.
- Accept correction and help repair harm when mistakes occur.

## Unacceptable behavior

Harassment, discrimination, threats, sexualized attention, deliberate
intimidation, doxxing, publication of private information, and sustained
disruption are not tolerated in project spaces.

## Enforcement

Report conduct concerns privately to the repository owner. Reports will be
reviewed promptly and handled as confidentially as practical. Maintainers may
edit or remove content, issue warnings, temporarily restrict participation, or
permanently exclude participants when necessary to protect the community.

This policy applies to repositories, issues, reviews, discussions, community
channels, events, and situations where someone is representing the project.
